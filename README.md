# 🧮 Numerical Analysis with MATLAB

這個專案展示了數學系專業中，如何將抽象的數學公式轉化為高效的電腦運算。

## 🚀 核心功能
- **Root Finding**: 包含牛頓法 (Newton's Method) 與 割線法 (Secant Method)。
- **Integration**: 辛普森準則 (Simpson's Rule) 的數值實踐。
- **Visualization**: 利用 MATLAB 強大的繪圖功能，將收斂過程視覺化。

## 📊 執行結果
> *圖說：展示了演算法在 5 次迭代內快速收斂至目標值。*

## 🛠️ 使用工具
- **Language**: MATLAB R2023b
- **Toolbox**: Optimization Toolbox
